## üèóÔ∏è Project Overview

Flow Lens is a code architecture visualization tool that analyzes codebases and generates interactive visualizations showing the relationships between components, modules, and functions.

### Key Components

- **Backend Analyzer** - Python-based codebase analysis engine
- **Frontend Explorer** - React-based visualization interface
- **Response JSON** - Structured output containing nodes and edges
